# Finstagram Profile Card
Responsive profile card built with HTML and CSS using Flexbox


## Overview

This project is a responsive social profile card created using HTML and CSS. The design presents a fish themed user profile with a background image, profile photo, statistics, and a simple password input section.

The goal of this project was to practice layout, positioning, and Flexbox centering techniques while organizing CSS for readability and maintainability.

## Features

* Full-page background image
* Centered profile card using Flexbox
* Header image with overlapping circular profile picture
* Social statistics section (followers, likes, photos)
* Password input and action button
* Clean and organized CSS structure

* HTML5

* Flexbox
